    Creating Blinky Project

    The easiest way to use FreeRTOS is to start with one of the pre-configured demo application projects by select “File - New - C Project” choose RISC-V Blinky Demo project. Select Toolchains: riscv-rvdn-elf and enter Project name. Then click Next.

    Enter Author then click Next.

    Select RISC-V FreeRTOS Project configuration by default with 2 modes: release/debug > Then Finish
